

Клавиша выбора готовых предустановок (пресетов)
(находится под клавишей Volume)

ПРЕСЕТЫ


1.

Reference Classical-

Использует 4 установки обработки структуры сигнала ,
для точного и более тонкого сохранения
спектрального баланса классической музыки.

.................................................
2.
Reference Jazz-

Пресет создает более мягкий частотный баланс, 
для точного сохранения ньюансов при контроле 
над резкими звуками.

...........................................

3.
Reference Low CPU-

Эталонный пресет- 
откалиброван с высокой точностью предустановки. 
Подходит для всех типов аудио.


Даже при сглаживании черезмерного уровня динамики,
сохраняет первоначальное ощущение естественности звучания.
Этот пресет с низким уровнем использования процессора. 
Идеально подходит для ноутбуков
 при работе от батареи.

....................

4.
Reference Movies-

Это легкая предустановка. 
Предназначенная для просмотра художественных фильмов.


Сохраняет диалог понятным, но держит контроль 

над слишком громкими звуками.

........................

5.

Reference Settings-

Это откалиброванный сбалансированный пресет.

Подходит для всех типов звукового материала.


Сохраняет первоначальную динамику. 
Но контролирует её через мерный уровень.
